# Third-party notices

## TwemojiCountryFlags.woff2

Country-flag glyphs are rendered with the **Twemoji Country Flags** web font
(`TwemojiCountryFlags.woff2`), bundled so flag emoji display on platforms whose
system fonts lack them (notably Chrome on Windows).

The font is built from **Twemoji**, the emoji artwork created by Twitter, Inc.
and other contributors.
